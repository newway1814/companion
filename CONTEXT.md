# Companion Context

## Language

**Companion** is the repository name for the product currently described as InterviewArena.

**InterviewArena** is a realtime AI interview practice product for students and job seekers.

**Practice session** is a user-facing interview simulation built from a resume, target job description, interview configuration, adaptive questions, transcript, scoring, and report.

**Pressure Mode** changes interviewer behavior and pacing while staying professional, respectful, and safe.

**Performance report** is the post-session dashboard that explains scores, strengths, weaknesses, transcript highlights, rewritten answers, and practice recommendations.

## Relationships

- A user can run many practice sessions.
- A practice session uses one resume and one target job description.
- A practice session produces questions, answers, scores, transcript highlights, and one performance report.

## Guardrails

- The product is for interview practice and coaching, not hiring decisions.
- Do not rank real candidates for employers.
- Do not evaluate protected traits or facial features.
- Keep feedback grounded in the user's resume, job description, and answers.
- Store sensitive resume and transcript data only when necessary.
